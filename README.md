# XMGTV

直播推流项目实践

说明:
由于同时支持模拟器和真机的`IJKMediaFramework.framework`大于100M,上传失败.
所以在IJKMediaFramework目录中分别保存了模拟器和真机的两种.framework,请使用相应的库文件即可运行成功.

如需合并库请参考教程
[打包合并iOS的ijkplayer静态库](http://www.jianshu.com/p/9f795749df4d)
